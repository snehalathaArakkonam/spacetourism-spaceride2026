## 🌌 Project Overview

**SPACERIDE2026** is an immersive, futuristic space tourism web platform designed to ignite curiosity and inspire the next generation of space explorers. Built entirely with pure HTML5, SCSS, and Vanilla JavaScript (no frameworks!), this interactive website brings the cosmos to life through engaging games, educational content, and stunning visual experiences.

### 🎯 The Problem We Solve
Space education often feels distant and inaccessible. Kids and students struggle to visualize space concepts, and existing resources lack interactivity. SPACERIDE2026 bridges this gap by creating a playful, educational space experience that feels like a real space tourism portal.

### ✨ Why It Exists
- To make space science fun and accessible for school kids (8-15 years) and college students
- To celebrate India's space achievements through the dedicated ISRO Corner
- To provide an interactive learning platform that combines education with entertainment
- To showcase modern web development techniques without relying on heavy frameworks

## | Layer | Technology | Purpose | Key Features Used |
|-------|------------|---------|-------------------|
| **Markup** | HTML5 | Semantic structure | Canvas API, Semantic tags, Forms, LocalStorage API |
| **Styling** | SCSS (Sass) → CSS | Advanced styling | Variables ($neon-blue, $dark-bg), Mixins (@mixin flex-center, glass-card), Nesting, Keyframe animations, CSS Grid, Flexbox, Backdrop-filter |
| **Interactivity** | Vanilla JavaScript | Core functionality | DOM manipulation, Canvas API (2D context), LocalStorage API, Event listeners, RequestAnimationFrame, Audio API |
| **Icons** | FontAwesome 6 | Visual enhancements | 500+ icons, Rocket icons, Planet icons, Social media icons |
| **Typography** | Google Fonts | Custom fonts | Orbitron (futuristic display), Exo 2 (clean body text) |
| **Animations** | CSS3 + JavaScript | Visual effects | CSS Keyframes (float, shoot, ticker), JavaScript Canvas animation, CSS Transitions, Glow effects |
| **Deployment** | Netlify | Hosting | Continuous deployment, HTTPS, Custom domain |
| **Version Control** | Git & GitHub | Source control | Repository management, Issue tracking |
| **Code Editor** | VS Code | Development | Live Server, SCSS compilation |

### 📦 No External Frameworks Used
- ❌ No React, Vue, or Angular
- ❌ No Bootstrap or Tailwind CSS
- ❌ No jQuery
- ✅ 100% Pure Vanilla Code

### 🎯 Why This Tech Stack?
| Choice | Reason |
|--------|--------|
| **Pure HTML5** | Maximum compatibility, semantic meaning, SEO friendly |
| **SCSS** | Maintainable styles with variables and nesting, compiled to lightweight CSS |
| **Vanilla JS** | No framework overhead, full control, faster load times |
| **Canvas API** | Smooth custom animations (stars, solar system, rocket trails) |
| **LocalStorage** | Persistent user achievements without backend |
| **FontAwesome CDN** | High-quality icons without image assets |
| **Google Fonts** | Modern typography with minimal load impact |
